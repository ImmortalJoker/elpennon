Enmarcaciones El Peñón is a small static Astro marketing site for a custom framing shop in Puerto de la Cruz. The app is built with Astro 6, TypeScript, Tailwind CSS v4, and i18next for client-side language switching. The main page is assembled in src/pages/index.astro from components in src/components (Hero, Gallery, Calculator, MapContact, Footer, LanguageSwitcher). Global theme colors and fonts live in src/styles/global.css. Static imagery is stored in public/, with display-optimized copies currently under public/optimized/.
