Useful commands for this repo:
- `npm install` to install dependencies
- `npm run dev -- --host 127.0.0.1 --port 4321` to run the local dev server
- `npm run build` to build the static site into dist/
- `rg <pattern> src` to search source files quickly
The environment is Darwin/macOS, so tools like `sips`, `ls`, `sed`, `open`, and `pbcopy` are available.
